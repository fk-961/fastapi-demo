# Expense Analytics FastAPI Demo
Mini FastAPI backend for expense tracking and analytics.

## Features
- GET all expenses
- POST new expenses
- DELETE expenses
- TOTAL spending analytics
- Spending by category
- User expense filtering
- JSON persistence

## Data Model
```json
{
    "amount" : 120,
    "category" : "food",
    "user" : "John" 
}
```
